Μία σύντομη περιγραφή

Στα πλαίσια του μαθήματος Προχωρημένα Θέματα Τεχνολογίας και Εφαρμογών Βάσεων Δεδομένων αναπτύξαμε μια εφαρμογή οπτικοποίησης δεδομένων που αφορούν χώρες και χρονολογίες σχετικά με 12 δείκτες που θα αναφέρουμε παρακάτω.Ο χρήστης θα είναι σε θέση να επιλέξη διάφορους συνδιασμούς των παραπάνω και να αντλήσει μεσο της οπτικοποίησης του αποτελέσματος διάφορα συμπεράσματα.Επίσης η μέθοδος οπτικοποίησης του αποτελέσματος μπορεί να επιλεχθεί χωρίς κάποιον περιορισμό.

Eπιλογές οπτικοποιησης
  • Scatter plot
  • Line plot
  • Bar chart
  • many more...
Περιγραφή βάσης,απο τι απολείται;
  • 12 χώρες.
  • 12 δείκτες.
  • Δεδομένα μεταξυ χρονολογιών-δεικτών για κάθε χώρα.
  • Δεδομένα χρονολογιών(σε ποια πενταετία και δεκαετία ανήκει η καθεμία).
